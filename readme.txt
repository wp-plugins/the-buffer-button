== Description ==

Add the buffer button to your blog to let people share your content on Twitter and Facebook seamlessly. They can share right away or at a better time using Buffer.
For more informations: http://bufferapp.com/extras/button

== Installation ==

1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. If you want to change the settings, click Settings->The Buffer Button and change the settings there.
